# Go Offer Curator & Learning Platform

Полностью обновленная интерактивная SaaS-панель для кураторов программы акселерации Go Offer. Компонент полностью переписан на **Tailwind CSS**, а также расширен новым разделом **«Обучение»** с графиками и интерактивными чек-листами.

## Разделы платформы
1. **Обучение (Новый)** — аналитика успеваемости менти, тренды активности по дням недели на графиках `recharts` и интерактивный чек-лист по модулям.
2. **Тарифы** — интерактивное сравнение условий всех пакетов услуг (Take All, Take All+, VIP, Comeback).
3. **База знаний** — фильтруемый справочник внутренних регламентов, гайдов и протоколов с удобной навигацией.
4. **AI-помощник** — генератор ответов менти в Tone of Voice компании.

## Как запустить локально
1. Распакуйте архив.
2. Установите зависимости: `npm install`
3. Запустите dev-сервер: `npm run dev`
4. Соберите проект: `npm run build`
